# Capsule-based-Fecal-Microbiota-Transplantation-for-Chronic-Insomnia
 Data analysis code for the registered randomized controlled trial “The Safety and Efficacy of Fecal Microbiota Transplantation in Patients with Chronic Insomnia Disorder” (ClinicalTrials.gov Identifier: NCT05917379).
## Code and Reproducibility

This repository contains the analysis code used for the registered randomized controlled trial 
“The Safety and Efficacy of Fecal Microbiota Transplantation in Patients with Chronic Insomnia Disorder” 
(ClinicalTrials.gov Identifier: NCT05917379).

All analyses can be reproduced using the publicly available dataset hosted on Zenodo.

De-identified individual participant data are available at Zenodo (https://doi.org/10.5281/zenodo.18136081), 
in accordance with institutional ethics approval and data-sharing regulations.
